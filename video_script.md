# Script Video Pengenalan IZQuran

**Durasi Perkiraan:** 2 - 3 Menit
**Visual:** Tampilan layar utama (localhost/IZQuran) yang direkam (Screen Record).

---

## 1. Pembukaan (Sapaan & Perkenalan)
*(Layar menampilkan halaman awal "Selamat Datang" dengan judul IZQuran)*

**Narasi:**
> "Halo, Assalamualaikum warahmatullahi wabarakatuh teman-teman developer dan penggiat Al-Quran semuanya! 
> 
> Kenalin, saya Asrianor. Pada kesempatan kali ini, saya mau *showcase* sekaligus memperkenalkan sebuah *mini project* open-source yang baru saja saya bangun, namanya **IZQuran**."

---

## 2. Penjelasan Konsep Aplikasi & Teknologi
*(Kursor bergerak perlahan memperlihatkan poin-poin fitur di layar)*

**Narasi:**
> "IZQuran ini adalah aplikasi interaktif anti-ribet berbasis web yang saya buat khusus untuk membantu melatih dan menguji hafalan Al-Quran kita. 
> 
> Menariknya, aplikasi ini dibangun murni menggunakan HTML biasa, Javascript *Client-Side*, dan CSS Vanilla bernuansa *Glassmorphism*. Jadi sangat ringan, cepat, tanpa butuh *database server* sama sekali! Kita murni 'menarik' datanya langsung menggunakan Endpoint API publik gratis dari **EQuran.id**."

---

## 3. Demo Ringkas Fitur-Fiturnya
*(Klik tombol "Mulai Sekarang", lalu ketik angka "78" pada input Surah dan klik Muat Surah)*

**Narasi:**
> "Oke, mari kita coba langsung. Begitu masuk, kita tinggal ketik nomor surah yang ingin dihafal, misalnya surah ke-78, An-Naba, lalu klik muat. Nah, di sini ada 3 mode utama."

*(Pilih Mode Persiapan, lalu coba ubah-ubah Layout Dropdown dari "Daftar Biasa" ke "4 Kuadran" atau "Kiri-Kanan" pada panel pengaturan)*

> "**Pertama, Mode Persiapan.** Di mode ini kita bisa _mereview_ bacaan. Yang bikin beda, saya sudah bikinkan fitur *Konfigurasi Layout* dan *Filter Rentang Ayat*. Jadi kalau kalian tipe visual yang gampang hafal kalau ayatnya dikelompok-kelompokkan ke dalam kotak per 5 ayat, dibikin kiri-kanan, atau bahkan 4 kuadran, itu semua bisa diatur di sini! Dan oh iya, di setiap ayat juga sudah ada pemutar _audio murottal_-nya."

*(Kembali ke atas, ganti pilih "Mode Potongan Awal". Perlihatkan UI flashcard)*

> "**Kedua, Mode Potongan Awal.** Ini semacam teknik *flashcard*. Aplikasi cuma ngasih tahu *kata pertama* atau potongan depan dari sebuah ayat. Tugas kita adalah menebak kelanjutannya secara lisan, baru kita klik *'Tampilkan Ayat Penuh'* untuk mencocokkan."

*(Lalu masuk ke "Mode Angka Dulu", tunjukkan urutan acak)*

> "**Ketiga, Mode Angka.** Ini versi *Hardcore*-nya. Yang muncul di layar murni cuma nomor urutan ayatnya saja. Kalau masih agak lupa, kita bisa ngintip kata pertamanya, baru melihat versi full-nya.
> 
> Kita juga bisa _setting_ urutannya mau diacak (Random), atau *Reversed* (diulang dari akhir ke awal) biar hafalan kita makin kuat secara *Random Access Memory*."
